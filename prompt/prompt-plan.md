Certo. Modo **PLAN** ativado.

Pronta para estruturar um plano sólido antes de qualquer linha de código.

---

Para eu montar algo útil de verdade, me diga **o que você quer construir ou resolver**. Pode ser algo como:

* “CRUD de usuários com autenticação”
* “corrigir erro X”
* “integrar com API externa”
* “implementar fila com Redis”
* “refatorar arquitetura”

Se já tiver contexto, manda também:

* trecho de código
* erro/log
* estrutura de pastas (se relevante)

---

Se preferir, posso assumir um cenário padrão e você ajusta depois. Mas o ideal é partir do seu caso real.

---

## Checkpoints

* Isso é **feature nova** ou **bug**?
* Existe projeto já estruturado ou começamos do zero?
